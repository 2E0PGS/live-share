Share your context, not just your screen

Frequent communication is critical to team success, so you need collaborative tools that are optimized for your personal workflow. Project Cascade lets you share your project context rather than just your screen. You get a bi-directional, instant, and familiar way to jump into opportunistic "ad-hoc pair programming" in your tool without being forced to look at the same file, viewport, etc.

The Project Cascade private preview currently supports these tools and platforms:
  • Visual Studio - ASP.NET/ASP.NET Core and Node.js projects
  • Visual Studio Code - ASP.NET Core and Node.js projects

Collaboration that is simple, yet comprehensive

Problem solving requires rapid collaboration and ensuring everyone is on the same page from the start. Often exchanging code snippets and error messages doesn't provide the full story which slows you down. With Project Cascade, you can collaboartivley edit and debug seamlessly and securely on a shared working copy of a project using your workflow and the comfort of your IDE/editor.

With Project Cascade, you can instantly share access to the following resources:
  • Your project's entire codebase
  • Application build output
  • Debugging sessions, including shared breakpoints
  • Terminal instances
  • Locally running servers or services
